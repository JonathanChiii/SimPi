# SimPi

## Requirement
 - python >= 3.5

## Set up
Update apt
```
sudo apt-get update -y
```

Install python 3.7
```
sudo apt install python3.7
```
Install websockets
```
sudo apt-get install -y python3-websockets
```

Run server.py
```
python3 server.py
```

Run index.html
